QUESTION 4: "Print output: var x = 21; var girl = function () { console.log(x); var x = 20; }; girl ();"

The output will be undefined because the value was not returned and also a variable has not been assigned a value. the first scope will look at scope level variables whether it is local or global level and if the variable is defined 2 times in global scope and also the local scope in the function then function takes the precedence local scope variable and if we use the variable before declaration then function will execute console statement .

the output is undefined and 21 , here girl is not defined. the first scope will look at scope level variables whether it is local or global level and if the variable is defined 2 times in global scope and also the local scope in the function then function takes the precedence local scope variable and if we use the variable before declaration then function will execute console statement .
